Origami Repo Data is an API which can be used to get information about repositories which contain an origami.json file. These could be components, services, image sets, and Node.js modules. Inclusion in this API does not mean a module is supported by the Origami team.

Criteria for inclusion

To be included in Origami Repo Data, a repository must meet the following criteria. When these criteria are met, the repo will be automatically ingested into this service.

  • MUST reside in the Financial-Times GitHub organisation
  • MUST contain a valid origami.json file (see the spec for more information)
  • MUST have at least one git tag which complies with semantic versioning
